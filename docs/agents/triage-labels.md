# 分流标签

工程类 skill 使用五个标准 triage 角色。本文档把这些角色映射到本仓库问题跟踪系统中实际使用的 label 字符串。

| mattpocock/skills 中的角色 | 本仓库 label | 含义 |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | 需要维护者评估 |
| `needs-info` | `needs-info` | 等待报告者补充信息 |
| `ready-for-agent` | `ready-for-agent` | 描述完整，可以交给 AFK agent |
| `ready-for-human` | `ready-for-human` | 需要人类实现 |
| `wontfix` | `wontfix` | 不会处理 |

当 skill 提到某个 triage 角色时，使用上表对应的 label 字符串。

如果 GitHub 仓库以后采用不同 label 词汇，只修改右侧映射列。
